🎮 Connect Four Game (React)

A fully interactive Connect Four game built using React. This project demonstrates core frontend concepts like state management, component structure, and game logic implementation.

✨ Features:
- 👥 Two-player mode with custom player names
- 🎨 Player color selection
- 🧠 Win detection (horizontal, vertical, diagonal)
- 🔁 Turn-based gameplay
- 🏆 Live scoreboard
- 🔄 Next round without resetting scores
- ⚡ Built completely without backend (frontend-only)

🛠 Tech Stack:
- React (useState)
- JavaScript
- HTML5 & CSS3

🚀 How to Run:
1. Clone the repository
2. Run `npm install`
3. Run `npm start`

📌 Future Improvements:
- AI opponent 🤖
- Local storage for saving scores 💾
- Multiplayer support 🌐
- Animations for disc drop 🎨

This project is ideal for beginners learning React and game logic implementation.
